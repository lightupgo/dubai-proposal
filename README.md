# Eight Days. One Yes.

Gez and Hetty’s Dubai proposal journey, remembered through photographs and films from 4–11 October 2019.

View the public site at [lightupgo.github.io/dubai-proposal](https://lightupgo.github.io/dubai-proposal/).

## Organising memories

Original materials stay local inside `materials/YYYY-MM-DD/`. A date can contain numbered folders such as `1 - Dubai Airport` and `2 - Atlantis The Palm`; these become ordered sections on that day’s page. Dates without numbered folders keep the original single-story layout.
